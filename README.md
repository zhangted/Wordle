# Wordle


## Building and running on localhost

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run build-prod
```

To create a development build:

```sh
npm run build-dev
```

## Running local dev server

```sh
npx webpack serve --mode production
```

## Testing

To run unit tests:

```sh
npm test
```
